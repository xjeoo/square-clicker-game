# 🟦 Square Clicker - 2D Arcade Game

> A fast-paced 2D arcade game built with C++ and SFML. I developed this project in a single day as a personal challenge to dive into programming.

## 🎮 How to Play
* Squares of different sizes and colors fall continuously from the top of the screen.
* Each color/size represents a specific difficulty level and deals a different amount of damage to your HP if it hits the ground.
* **Goal:** Click the squares to destroy them before they reach the bottom!
* **Score:** Earn points for every square successfully destroyed.
* **Game Over:** If your HP reaches 0, the game ends (but you can restart and try to beat your high score).

## 🛠️ Tech Stack
* **Language:** C++
* **Graphics Library:** SFML 2.6
* **IDE:** Visual Studio

## 🚀 How to Build and Run Locally

### 1. Clone this repository
```bash
git clone <link-of-this-repository>
```
### 2. SFML Setup
Make sure you have SFML installed and configured for Visual Studio. If you haven't set it up before, you can follow the [official SFML Tutorial](https://www.sfml-dev.org/tutorials/2.6/start-vc.php) for Visual Studio.

### 3. Build the Game
* Open the SquareClicker.sln file in Visual Studio.

* Make sure your build configuration matches your SFML setup (e.g., Debug or Release, x64).

* Build and run the project by pressing F5.
